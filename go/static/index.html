🌍 时区处理演示 API

SAAS多租户时区处理解决方案

📋 可用的 API 端点

GET /api/health
健康检查接口
GET /api/docs
API 文档和使用说明
GET /api/timezone/demo
时区转换演示
GET /api/timezone/merchants
获取商户列表
GET /api/timezone/orders
获取订单数据（支持时区转换）
GET /api/timezone/analysis
获取订单分析数据
GET /api/timezone/compare
时区对比功能

🚀 快速开始

这是一个演示多租户时区处理的后端 API 服务。所有接口都支持跨域访问，可以直接在浏览器中测试。

示例请求：

  • 检查服务状态
  • 查看详细文档
  • 获取商户列表
